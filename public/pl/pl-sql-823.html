Diagnostyka Bazy Danych: SQL Server Error 823

Msg 823, Level 24, State 2
The operating system returned error 21(The device is not ready.) to SQL Server during a read at offset 0x000000000ae000 in file 'C:\Data\Database.mdf'.

Analiza Inżynieryjna:

Błąd 823 jest błędem krytycznym, który zazwyczaj wskazuje na problem z podsystemem I/O lub uszkodzenie fizyczne nośnika. W przeciwieństwie do błędów logicznych, błąd ten oznacza, że SQL Server nie może ukończyć operacji odczytu/zapisu na dysku.

  • Sprawdzenie sum kontrolnych (Checksum) stron bazy danych.
  • Naprawa przerwanych łańcuchów stron (Page Chain).
  • Rekonstrukcja tabel bez konieczności przywracania z uszkodzonych kopii zapasowych.
