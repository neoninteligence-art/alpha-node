Kritická chyba SQL Serveru: ID 824

Pokud se setkáte s chybou 824 (Logical Consistency I/O Error), znamená to, že SQL Server úspěšně přečetl stránku z disku, ale data na stránce jsou logicky poškozená. Tento problém často signalizuje selhání diskového subsystému nebo poškození souborového systému NTFS.

Doporučený postup opravy:

  • Spusťte DBCC CHECKDB pro zjištění rozsahu poškození.
  • Zkontrolujte logy Windows (Event Viewer) na přítomnost chyb disku.
  • Pokud nemáte platnou zálohu, použijte specializovaný nástroj pro rekonstrukci stránek MDF.

Ruční oprava pomocí REPAIR_ALLOW_DATA_LOSS může vést k vymazání důležitých záznamů. Pro zachování integrity dat doporučujeme extrakci schémat a dat pomocí profesionálního obnovovacího motoru.
